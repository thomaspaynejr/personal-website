# HOW TO PUSH YOUR PERSONAL WEBSITE TO GITHUB

Follow these steps to link your local project to a new GitHub repository:

1. CREATE A REPOSITORY ON GITHUB:
   - Go to github.com and click the '+' icon -> 'New repository'.
   - Name it 'personal-website' (or your preferred name).
   - Keep it Public or Private as you wish.
   - DO NOT initialize with a README, license, or gitignore (we already have these).
   - Click 'Create repository'.

2. LINK YOUR LOCAL PROJECT (RUN IN YOUR TERMINAL):
   Copy and paste these commands into your terminal while inside the project folder:

   # Add the remote origin (replace [USERNAME] with your GitHub username)
   git remote add origin https://github.com/[USERNAME]/personal-website.git

   # Rename branch to main (just in case)
   git branch -M main

   # Push your code
   git push -u origin main

3. SUCCESS:
   Refresh your GitHub page and you will see your "Yeezy-inspired" dashboard and code live!

---
Note: If you haven't set up your GitHub credentials on this machine yet, 
GitHub may ask you to sign in or use a Personal Access Token (PAT).
